# API Endpoints and Templates

## FRED API

Base:

```text
https://api.stlouisfed.org/fred/series/observations
```

Template:

```text
https://api.stlouisfed.org/fred/series/observations?series_id={SERIES_ID}&api_key={FRED_API_KEY}&file_type=json&observation_start=2005-01-01
```

Example (`FEDFUNDS`):

```text
https://api.stlouisfed.org/fred/series/observations?series_id=FEDFUNDS&api_key=YOUR_KEY&file_type=json&observation_start=2005-01-01
```
