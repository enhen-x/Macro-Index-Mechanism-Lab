# 全球层 + 国家层的受迫振动建模方案

## 1. 目标

在不引入“经济区层”的前提下，建立统一的两层结构：

- 全球层（Global Layer）：反映全球共同冲击
- 国家层（Country Layer）：反映各国本地冲击与政策特征

用于检验模型在不同国家市场中的普适性与异质性。

本方案强调一个前提：模型基于控制理论，输入变量应优先选择“可被国家机构主动调节”的政策工具。

## 2. 核心思想

每个国家市场都使用同一个二阶受迫阻尼结构，但输入项分为两组：

1. 全球共同冲击 `U_global,t`
2. 国家本地冲击 `U_country,i,t`

对第 `i` 个国家市场的主体变量 `Y_i,t`（如指数 gap）建模：

```math
m_i \ddot{Y}_{i,t} + c_i \dot{Y}_{i,t} + k_i (Y_{i,t} - x0_i)
= \beta_{g,i}^{\top} U_{global,t} + \beta_{c,i}^{\top} U_{country,i,t} + \epsilon_{i,t}
```

其中：

- `m_i, c_i, k_i`：国家 `i` 的惯性、阻尼、回复力参数
- `\beta_{g,i}`：国家 `i` 对全球冲击的暴露系数
- `\beta_{c,i}`：国家 `i` 对本地冲击的暴露系数

## 3. 主体变量（被解释变量）

每国选代表性宽基指数，定义：

```math
Y_{i,t} = \log(Index_{i,t}) - trend_{i,t}
```

建议：

- 频率统一月频
- `trend_{i,t}` 可先用 HP filter（λ=129600）

## 4. 输入变量设计（政策可控优先）

## 4.1 全球层变量 `U_global,t`（外生共同冲击）

建议最小集合：

1. `US10Y`：美国 10Y 国债收益率变化
2. `DXY`：美元指数变化
3. `VIX`：全球风险偏好变化
4. `Brent`：原油价格变化

处理口径：

- 使用一阶差分或对数收益
- 训练集内标准化（z-score）

## 4.2 国家层变量 `U_country,i,t`（政策可控变量）

每个国家统一字段，但数据取本国值。优先使用可被央行/监管/财政部门调节的变量：

1. 政策利率变化 `ΔPolicyRate_i,t`（核心控制量）
2. 央行流动性投放/回笼 `ΔLiquidityOps_i,t`（OMO/MLF/再贷款等）
3. 准备金率变化 `ΔRRR_i,t`（适用于有该工具的国家）
4. 信贷扩张变化 `ΔCredit_i,t`（社融/银行信贷增速）
5. 监管或政策事件哑变量 `PolicyEvent_i,t`

可选扩展：

- 本币兑美元变化 `ΔFX_i,t`（部分国家属于政策可部分影响）
- 财政支出增速（有稳定月度代理时）

不建议作为“国家层控制输入”的主变量：

- VIX、油价、DXY 等全球风险因子（应归入全球外生冲击）

## 5. 数据处理流程（统一）

1. 全部序列对齐到月末
2. 指数转 `log` 并构造 `Y_i,t`
3. 输入变量做差分/收益化
4. 缺失值按统一规则处理（短缺口 ffill，长缺口剔除）
5. 训练/验证/测试按时间切分（如 70/15/15）
6. 所有标准化仅在训练集拟合，再应用到验证与测试

## 6. 离散化与估计

这里采用“半步插值 + 中心差分”来构造速度与加速度。

### 6.1 时间索引与插值定义

设观测序列为月频 `Y_{i,t}`，步长 `Δt=1`（也可保留符号 `Δt` 统一表示）。

半步点用线性插值：

```math
Y_{i,t+\frac12}=\frac{Y_{i,t+1}+Y_{i,t}}{2}, \quad
Y_{i,t-\frac12}=\frac{Y_{i,t}+Y_{i,t-1}}{2}
```

据此定义中心差分速度：

```math
v_{i,t}
=\frac{Y_{i,t+\frac12}-Y_{i,t-\frac12}}{\Delta t}
=\frac{Y_{i,t+1}-Y_{i,t-1}}{2\Delta t}
```

再定义半步速度：

```math
v_{i,t+\frac12}=\frac{Y_{i,t+1}-Y_{i,t}}{\Delta t},\quad
v_{i,t-\frac12}=\frac{Y_{i,t}-Y_{i,t-1}}{\Delta t}
```

加速度使用中心差分：

```math
a_{i,t}
=\frac{v_{i,t+\frac12}-v_{i,t-\frac12}}{\Delta t}
=\frac{Y_{i,t+1}-2Y_{i,t}+Y_{i,t-1}}{\Delta t^2}
```

当 `Δt=1` 时，上式简化为你提出的口径：

- `v_{i,t}=(Y_{i,t+1}-Y_{i,t-1})/2`
- `a_{i,t}=Y_{i,t+1}+Y_{i,t-1}-2Y_{i,t}`

### 6.2 回归估计式（可直接落地）

把连续模型写成离散可估计形式：

```math
\frac{Y_{i,t+1}-2Y_{i,t}+Y_{i,t-1}}{\Delta t^2}
= -\frac{c_i}{m_i}\frac{Y_{i,t+1}-Y_{i,t-1}}{2\Delta t}
  -\frac{k_i}{m_i}(Y_{i,t}-x0_i)
  +\tilde{\beta}_{g,i}^{\top} U_{global,t}
  +\tilde{\beta}_{c,i}^{\top} U_{country,i,t}
  + e_{i,t}
```

估计时建议：

1. 有效样本索引取 `t=2,...,T-1`（首尾点因中心差分不可用而丢弃）
2. 先做单市场 OLS 基线，再做滚动窗口估计（时变参数）
3. 最后做联合估计（可选：分层贝叶斯）

## 7. 普适性检验框架

## 7.1 单国有效性

- 每国样本外误差（MSE/MAE）
- 方向准确率（Directional Accuracy）
- 阻尼类型判别：
  - `c_i^2 < 4m_i k_i` 欠阻尼
  - `c_i^2 = 4m_i k_i` 临界阻尼
  - `c_i^2 > 4m_i k_i` 过阻尼

## 7.2 跨国可比性

- 参数符号一致性（特别是 `c_i > 0, k_i > 0`）
- 全球暴露系数 `\beta_{g,i}` 的国家排序是否符合直觉
- 本地冲击系数 `\beta_{c,i}` 是否体现政策与制度差异

## 7.3 稳健性

- 不同样本窗口（危机前后）重复估计
- 替换变量口径（例如 PMI 替换工业生产）
- 删除单一全球因子后的敏感性测试

## 8. 推荐国家集合（第一批）

建议先从 8 个市场起步：

1. 美国
2. 中国
3. 日本
4. 英国
5. 德国（或欧元区指数）
6. 印度
7. 巴西
8. 加拿大

理由：

- 覆盖发达与新兴
- 货币制度与政策风格差异明显
- 可检验“全球冲击 + 本地政策”框架的解释力

## 10. 实施优先级

1. 先跑美国单市场基线（数据完整、政策传导清晰）
2. 再加入中国做双市场对照（检验政策机制异质性）
3. 扩展到 8 国样本
4. 做滚动估计与危机窗口对比
5. 最后再考虑更复杂的联合估计方法

## 11. 最小可行变量集（MVP）

为快速启动，建议每国先使用以下 5 个国家层输入：

1. `ΔPolicyRate_i,t`
2. `ΔLiquidityOps_i,t`
3. `ΔCredit_i,t`
4. `PolicyEvent_i,t`
5. `ΔFX_i,t`（若本国汇率市场化程度高则可保留）

全球层保留 3 个外生项即可启动：

1. `ΔUS10Y_t`
2. `ΔDXY_t`
3. `ΔVIX_t`
