# MacroDynamics-System-ID

将股票市场指数建模为受迫阻尼动力系统。

中文 | [English](./README.en.md)

## 1. 项目概述

`MacroDynamics-System-ID` 是一个实验性量化研究项目，目标是把指数行为映射为二阶控制系统，并通过系统辨识与状态空间建模估计可解释参数。

核心思路：

- 将指数波动视作对宏观信号输入的动态响应。
- 用可辨识参数刻画市场行为：
  - `m`（惯性 / 质量）
  - `c`（阻尼）
  - `k`（刚度 / 均值回复强度）
- 构建可跨市场验证的统一框架（如沪深300、标普500、日经225）。

## 2. 理论模型

二阶受迫阻尼系统：

```math
m\frac{d^2x}{dt^2} + c\frac{dx}{dt} + k(x - x_0) = F_{macro}(t) + \eta(t)
```

符号说明：

- `x(t)`：指数水平（或对数价格等变换序列）
- `m`：惯性，表示价格趋势变化的阻力
- `c`：阻尼，表示政策约束、风险控制与微观结构摩擦
- `k`：刚度，表示价格向均衡位置回拉的强度
- `F_macro(t)`：由宏观变量构成的外部驱动力
- `\eta(t)`：不可观测噪声 / 残差扰动

## 3. 方法论

### 3.1 系统辨识（System Identification）

基于历史输入输出数据估计 `[m, c, k]` 及外部驱动效应：

- 输入：宏观信号（如 LPR、M2 增速、汇率指数、政策代理变量）
- 输出：指数收益或对数价格动态
- 候选方法：
  - 最小二乘 / 正则化回归
  - 卡尔曼滤波 / 扩展卡尔曼滤波
  - 滚动估计或分状态估计

### 3.2 状态空间表示（State-Space Representation）

将模型转化为适合滤波、预测与控制的状态空间形式：

```math
\begin{bmatrix}
\dot{x}_1 \\
\dot{x}_2
\end{bmatrix}
=
\begin{bmatrix}
0 & 1 \\
-\frac{k}{m} & -\frac{c}{m}
\end{bmatrix}
\begin{bmatrix}
x_1 \\
x_2
\end{bmatrix}
+
\begin{bmatrix}
0 \\
\frac{1}{m}
\end{bmatrix}
u(t)
```

其中：

- `x_1`：价格水平状态
- `x_2`：速度 / 动量状态
- `u(t)`：宏观输入信号

### 3.3 跨市场验证（Cross-Market Validation）

通过多市场对比检验参数稳定性与预测质量：

- 中国 A 股代表性指数
- 美国大盘指数
- 日本代表性指数

评估维度：

- 样本外拟合能力
- 转折点敏感度
- 制度/状态切换适应性

## 4. 路线图（Roadmap）

- [ ] Phase 1: 数据管线
  - 构建价格与宏观数据的标准化加载器
  - 处理频率对齐、缺失值与修订值
- [ ] Phase 2: 基线模型
  - 实现静态与滚动参数估计
  - 建立基线诊断与残差分析
- [ ] Phase 3: 自适应建模
  - 引入时变参数（如 EKF）
  - 支持政策冲击与状态切换敏感性分析
- [ ] Phase 4: 物理约束机器学习融合
  - 探索 PINNs 或混合约束建模
  - 与纯数据驱动方法做对照评估
